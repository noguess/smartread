# Smart Reader 用户使用指南

欢迎使用 **Smart Reader**！这是一个基于**艾宾浩斯记忆曲线**与**生成式 AI** 打造的智能英语阅读平台。本指南将帮助您充分利用平台的各项功能，实现“在阅读中高效记单词”的学习目标。

---

## 📚 1. 核心学习流程 (Quick Start)

Smart Reader 提倡 **"Learn by Reading" (在阅读中学习)** 的理念。建议您遵循以下闭环流程：

1.  **单词管理**：系统会自动管理您的词汇表。你也可以在“单词本”中查看或手动调整单词状态。
2.  **生成文章**：点击首页的“**开始今日阅读**”或前往文库。系统会根据您当前**急需复习**的单词，利用 AI 生成一篇独一无二的短文。
3.  **沉浸阅读**：在阅读过程中，点击高亮的核心词查看释义、例句和视频讲解。使用“句子分析”功能拆解长难句。
4.  **即时测验**：阅读完毕后，进行课后测验。系统会根据您的答题情况，自动更新单词的记忆状态（Mastered/Learning 等）。
5.  **复习回顾**：通过“统计”页面查看学习时长和进度，坚持每日打卡。

## 🌟 2. 功能详解

### 2.1 仪表盘 (Dashboard)
首页是您的学习指挥中心：
*   **学习概览**：查看今日已掌握单词数、待复习数及连续打卡天数。
*   **快捷入口**：
    *   **Start Review Session** (开始复习)：优先针对今日到期单词生成文章。
    *   **I'm Feeling Lucky** (手气不错)：随机抽取主题或单词生成文章。

### 2.2 单词本 (Vocabulary)
这里是您的词汇库，所有单词按记忆状态分类：
*   **New (新词)**：尚未学习或刚导入的单词。
*   **Learning (学习中)**：答错或系统判定需要加强记忆的单词。
*   **Review (复习中)**：已学习但需要定期回顾的单词。
*   **Mastered (已掌握)**：已通过多次测试，形成长期记忆的单词。

**特色功能**：
*   **搜索与筛选**：快速查找特定单词。
*   **手动修正**：如果您发现某个词被误判，可以手动修改其状态。

### 2.3 文库与文章生成 (Library)
*   **生成模式**：
    *   **智能推荐**：系统自动根据遗忘曲线挑选单词。
    *   **手动选择**：在“手动生成”模式下，您可以指定特定的单词或输入自定义主题（Topic）来生成文章。
*   **文章分级**：文章难度分为 **L1 (基础)**、**L2 (进阶)**、**L3 (挑战)**，您可以根据自己的水平进行调整。
*   **文库列表**：所有生成的文章都会自动保存。您可以随时回顾旧文章，并重新进行测试（"一文多测"）。

### 2.4 沉浸式阅读 (Reading Experience)
这是产品的核心界面，专为深度学习设计：
*   **核心词高亮**：文章涵盖的目标单词会高亮显示。
*   **Deep Dive (深度查词)**：点击任意单词（不仅仅是高亮词）唤起详情卡片：
    *   **释义与发音**：提供中英文释义及美式/英式发音。
    *   **视频例句**：(特色) 自动匹配 **YouTube** 或 **Bilibili** 上的真实语境视频片段，支持一键播放定位到单词出现的时间点。
    *   **Pin (置顶)**：您可以将最喜欢的视频例句“置顶”，下次查词时优先显示。
*   **Sentence Analysis (长难句分析)**：长按或选中长难句，AI 将为您：
    *   分析句子结构的“骨架”。
    *   拆解语法成分。
    *   提供精准翻译。

### 2.5 智能测验 (Quiz System)
阅读完成后，点击底部的“Start Quiz”开始测试：
*   **题型多样**：包含单选、填空、拼写、连线等多种题型，全方位检测记忆。
*   **自适应难度**：
    *   连续满分且反馈“简单” -> 解锁更高难度题型。
    *   连续错误且反馈“太难” -> 降低难度，保护学习信心。
*   **SRS 自动同步**：测试结果直接决定单词下一次复习的时间（基于 Intervals 算法）。

### 2.6 统计 (Statistics)
可视化您的进步轨迹：
*   **学习分布**：查看每天的学习时长和文章阅读量。
*   **记忆曲线**：监控单词总量的增长趋势。

### 2.7 设置 (Settings)
*   **外观**：支持浅色/深色模式及主题色切换。
*   **阅读偏好**：设置文章默认长度（Short/Medium/Long）。
*   **数据管理**：
    *   **重置进度**：输入管理员密码 (`admin`) 可清空所有学习记录（慎用）。
    *   **备份导出**：支持导出 JSON 格式的学习数据。

## 💡 3. 使用小贴士 (Tips)

1.  **善用 iPad**：本应用针对 iPad 竖屏和横屏均做了深度适配，是最佳的学习伴侣。
2.  **不要通过背单词表来学习**：请确信“在语境中记忆”是效率最高的方式。直接去阅读文章，在文章中遇到生词再去查，记忆效果倍增。
3.  **关注“Learning”列表**：这是您当前的薄弱环节，建议优先清理该列表中的单词。
4.  **视频源切换**：如果您身处国内，建议在设置中将视频源优先设置为 **Bilibili** 以获得流畅体验。
