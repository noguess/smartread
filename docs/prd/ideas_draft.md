# V5.0 功能想法草稿

以下是一些待讨论的功能想法：

1. 分数页展示做错的题，并给出错题分析

2. 添加做题时间的分布图，汇总数据后用饼状图来展示不同时间段的占比。

3. 增加不同难度的做题时间的趋势图，用来观测相同难度下做题时间的变化。

4.学习过程中，主动查询的单词自动记录到单词本中，状态置为“学习中”

5.阅读页增加划词查询功能

6.重构文章和试题模块。增加文库功能，记录所有文章的元数据。文章可使用大模型生成或用户自主上传。上传文章或生成文章后，自动记录到文库。当用户在文章页面点击测试后，使用大模型自动生成试题，一篇文章可进行多次测试，每次测试生成的试题不同。注意，可基于现有功能进行改造，不要进行系统重写。

7.视频列表增加手动置顶功能，允许用户手动将其认为最准确的视频置顶，后续默认播放置顶视频

8.增加长句分析功能，用户可划整句，系统自动分析并给出分析结果
